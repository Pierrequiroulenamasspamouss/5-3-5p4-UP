The folder Minionsparadise contains a Unity project on version Unity 5.3. Keep that in mind so that your code follows C# 3.0 structure, if you try to edit cs files inside. The game is called Minions Paradise, from EA. I have modified it so it no longer requires on .unity3d bundles but instead on local assets in Assets/resources/content. I also did other changes that are not critical. 



The folder SERVER contains a flask server of the game's private server. 
The folder DOC contains documentation of the project. Please ensure to add documentation to whatever you do that could deserve documentation. 

Do not hallucinate, and try to use only the tools you have available, or create python scripts that will ease the work.
