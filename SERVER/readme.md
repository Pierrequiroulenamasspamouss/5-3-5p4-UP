How to run :

Install flask



config.json has to be configured to redirect to the server you want. Make a copy of config.json.local and rename it into config.json to replace the original config.json if you want local or 
do the same for public by renaming config.json.public -> config.json(copy) -> config.json so that you keep the old .public and the .local files if you want to revert the changes
